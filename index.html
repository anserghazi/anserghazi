Runroute

Runroute is a route planner for at-home runners, but it can be used anywhere. Nationwide lockdowns have made it difficult for runners like myself to hit the gym, so Runroute is intended to add variety to people's home running routes and illuminate local landmarks. All routes begin and end at the location users specify, and each park they select is a waypoint for the route.

Runroute was built using HTML, CSS, Javascript, and uses Google Maps Javascript API, Directions API, Places API, Geolocation API.
